# SEO Tags for Remaining Pages

Quick reference for adding SEO meta tags to the 6 remaining tool pages.

## Template Structure

Add this right after `<meta name="viewport">` in each page's `<head>`:

```html
<!-- SEO Meta Tags -->
<title>[SEO-OPTIMIZED TITLE]</title>
<meta name="description" content="[150-160 char description with keywords]">
<meta name="keywords" content="[comma, separated, keywords]">
<meta name="author" content="QuickTools">

<!-- Open Graph / Facebook -->
<meta property="og:type" content="website">
<meta property="og:url" content="https://quicktools.io/[page-name]">
<meta property="og:title" content="[Social share title]">
<meta property="og:description" content="[Social share description]">
<meta property="og:image" content="https://quicktools.io/og-[page-name].png">

<!-- Twitter Card -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:url" content="https://quicktools.io/[page-name]">
<meta name="twitter:title" content="[Twitter title]">
<meta name="twitter:description" content="[Twitter description]">
<meta name="twitter:image" content="https://quicktools.io/og-[page-name].png">

<!-- Canonical URL -->
<link rel="canonical" href="https://quicktools.io/[page-name]">
```

## How to Add These:

1. Open each HTML file
2. Find the `<head>` section
3. Locate the line: `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
4. Add the SEO tags **right after** that line
5. Keep the existing `<link>` tags (fonts, styles) below
6. Save and commit

**Estimated time: 30 minutes for all 6 pages**

---

## After Adding:

1. Update sitemap.xml `<lastmod>` dates to today
2. Commit changes
3. Deploy to production
4. Submit updated sitemap to Google Search Console
5. Check pages render correctly with new tags

**Tools to verify:**
- View source (Ctrl+U) to see meta tags
- Google Rich Results Test: https://search.google.com/test/rich-results
- Facebook Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
